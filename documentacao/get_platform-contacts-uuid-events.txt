Instruções de uso
Essa API exibe, por padrão, um limite de 10 conversões por página.
Para visualizar todas as conversões, é necessário utilizar os Query Params para paginação, seguindo as regras abaixo:
Nome
Descrição
orderCampo a ser ordenado. O único valor aceito é "created_at"
directionOrdenação da lista. "asc" ou "desc", padrão é "asc"
pageNúmero atual da página
Exemplo de como consultar as conversões mais recentes:
JSON
https://api.rd.services/platform/contacts/{UUID}/events?event_type=CONVERSION&order=created_at:desc&page=1
