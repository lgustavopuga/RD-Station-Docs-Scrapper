Criar negociação
post https://api.rd.services/crm/v2 /deals
Este end-point permite a criação de uma nova negociação no RD Station CRM. É o ponto de partida para automatizar a entrada de novas oportunidades de venda, seja a partir de um formulário no site, de outro sistema ou de um evento de marketing.
Importante: Somente é permitido criar negociações com status ongoing.
